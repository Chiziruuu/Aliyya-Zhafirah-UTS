## 🚀 Cara Menjalankan di VS Code

### 1. Install Python
Pastikan Python 3.8+ sudah terinstall.
Cek dengan: `python --version`

### 2. Buka Folder Project di VS Code
```
File → Open Folder → pilih folder "project"
```

### 3. Buka Terminal di VS Code
```
Terminal → New Terminal  (atau Ctrl + `)
```

### 4. Buat Virtual Environment (direkomendasikan)
```bash
python -m venv venv
```

Aktifkan virtual environment:
- **Windows:**   `venv\Scripts\activate`
- **Mac/Linux:** `source venv/bin/activate`

### 5. Install Dependencies
```bash
pip install -r requirements.txt
```

### 6. Jalankan Aplikasi
```bash
python app.py
```

### 7. Buka di Browser
Buka: [http://127.0.0.1:5000](http://127.0.0.1:5000)
